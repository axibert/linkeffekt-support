# LinkEffekt — Support

LinkEffekt is an AUv3 audio effect plugin that streams stereo audio from any track directly into Ableton Live using Link Audio.

## Contact

For support, feedback, or bug reports:

## Requirements

- Ableton Live 12.4 or later on the same local network
- Any AUv3-compatible host app (AUM, GarageBand, Logic Pro, Loopy Pro, Drambo, Cubasis, etc.)
- iOS 17+ / macOS 14+
